Le site de Baldr

_______________________________________________________________________________________________________________________________________________________________________________________________________________

Baldr, aussi sous le nom de Balder, est un logiciel de prévention contre le plagiat. Inspiré par les recherches récentes dans le domaine de l'informatique, Cela permet de comparer les documents en utilsant uniquement les propriétés reliées a la compression des données.



Le nom du projet est tiré de la mythologie scandinave. Baldr (vieil islandais Baldr, latin Balderus) était un des Ases, le fils d'Odin et de Frigg. Il est le dieu de l'innocence.
